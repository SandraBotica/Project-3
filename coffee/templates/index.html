Coffee Data API

Welcome to the Coffee Data API. This API provides data about coffee imports, exports, prices paid to growers, and retail prices for various countries over the years. You can also access visualizations for each table through the provided links.

Available API Endpoints:

  • /api/v1.0/data/imports - Coffee imports data
  • /api/v1.0/data/exports - Coffee exports data
  • /api/v1.0/data/prices_paid_to_growers - Prices paid to growers data
  • /api/v1.0/data/retail_prices - Retail prices data
