Welcome to the Coffee Data API.

This API provides data about coffee imports, exports, prices paid to growers, and retail prices for various countries over the years 1990-2019.

This data was obtained from the International Coffee Organization https://www.ico.org/new_historical.asp

You can also access visualisations for each table through the provided links.

Available Data API Endpoints:

  • /api/v1.0/data/imports - Coffee imports data
  • /api/v1.0/data/exports - Coffee exports data
  • /api/v1.0/data/prices_paid_to_growers - Prices paid to growers data
  • /api/v1.0/data/retail_prices - Retail prices data
